# GameInputWrapper

C++/CLI wrapper for Microsoft.GameInput rumble support. Rumbles **all** connected Xbox gamepad devices.

**Build requirement:** Visual Studio 2022 with **Desktop development with C++** workload (includes C++/CLI). The project targets .NET Framework 4.8 and x64. Open the solution in Visual Studio and build with Configuration **Release** and Platform **x64**.
